# LimeChat "Colloquy Standard" Theme

## Install

```bash
cd ~/Library/Application\ Support/LimeChat/Themes
git clone git://github.com/Krinkle/limechat-theme-colloquy.git
ln -s limechat-theme-colloquy/ColloquyStandard.* .
```
